# TopoLab — AI 3D 拓扑低模评测工具

TopoLab 是一个面向游戏美术、外包验收和模型优化团队的 Web Demo。它把主观的“模型看起来不错”拆解为可解释的质量标准，并提供模型浏览、质量评分、双模型 PK 和数据分析能力。

## 产品思考

评测采用五项加权标准：轮廓还原 25%、拓扑流向 25%、面数效率 20%、变形质量 20%、UV / 法线 10%。分数服务于优化决策：系统同时展示风险等级、高频问题和建议动作。Demo 数据均为模拟数据，并已在界面中明确标注。

## 功能

- 评测标准：指标、权重、检查目标和样例均值
- 模型浏览：分类筛选、模型基础数据和各维度表现
- 质量评分：可调节五维评分、加权实时计算、评审意见
- 模型 PK：两组模型自由选择、维度对比和决策建议
- 数据分析：趋势、质量分布、高频问题和优化提升
- 响应式界面：适配桌面和移动端，支持键盘焦点与减少动画偏好

## AI 工具与开发过程

本项目使用 OpenAI Codex 完成。协作过程包括：理解题目与界定使用场景；设计可量化的低模评测标准；建立信息架构与交互流程；生成 React / TypeScript 与 CSS 实现；通过构建检查修复问题；最后部署为可访问网页。AI 主要承担产品方案推演、界面文案、代码实现与验证，人负责需求目标和最终判断。

## 本地运行

```bash
pnpm install
pnpm dev
```

然后访问终端显示的本地地址。

## 技术栈

- React 19 + TypeScript
- Vinext / Vite
- 原生 CSS（无图表或 3D 库依赖）
- Cloudflare Workers 兼容构建

## 说明

这是用于 Vibe Coding 评测的产品原型。当前模型视图为轻量化线框视觉模拟；真实产品可进一步接入 Three.js、glTF/FBX 解析、自动拓扑检测和团队项目数据源。
