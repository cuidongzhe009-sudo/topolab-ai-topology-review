# TopoLab — AI 3D 拓扑低模评测工具

TopoLab 是一个面向游戏美术、外包验收和模型优化团队的 Web Demo。它把主观的“模型看起来不错”拆解为可解释的质量标准，并提供模型浏览、质量评分、双模型 PK 和数据分析能力。

## 产品思考

评测面向游戏动画绑定，采用六项加权标准：轮廓还原 15%、变形边流 25%、绑定适配 25%、面数效率 15%、拓扑健康 10%、UV / 法线 10%。重点检查肩肘髋膝环线、权重缓冲带、极点位置、非流形与退化面。OBJ 本身不包含骨骼和蒙皮权重，因此“绑定适配”表示网格的可绑定性，最终仍需结合引擎动作测试。

## 功能

- 评测标准：指标、权重、检查目标和样例均值
- 模型浏览：连续上传一个或多个 OBJ，新模型会追加到本地模型库，不覆盖已有模型
- 网格分析：统计三角面、顶点、四边面比例、边界边、非流形、N-gon 与 UV/法线覆盖
- 质量评分：六维绑定导向评分、加权实时计算和专家复核意见
- 模型 PK：任意选择两个样例或本地上传模型，真实渲染并比较网格统计与六维得分
- 数据分析：趋势、质量分布、高频问题和优化提升
- 响应式界面：适配桌面和移动端，支持键盘焦点与减少动画偏好

## AI 工具与开发过程

本项目使用 OpenAI Codex 完成。协作过程包括：理解题目与界定使用场景；设计可量化的低模评测标准；建立信息架构与交互流程；生成 React / TypeScript 与 CSS 实现；通过构建检查修复问题；最后部署为可访问网页。AI 主要承担产品方案推演、界面文案、代码实现与验证，人负责需求目标和最终判断。

## 本地运行

```bash
pnpm install
pnpm dev
```

然后访问终端显示的本地地址。

## 技术栈

- React 19 + TypeScript
- Vinext / Vite
- 原生 WebGL OBJ 解析与交互式 3D 预览
- 原生 CSS（无图表或 3D 库依赖）
- Cloudflare Workers 兼容构建

## 说明

这是用于 Vibe Coding 评测的产品原型。上传文件只在浏览器本地解析，不会发送到服务器；刷新页面后本地模型列表会清空。真实产品可进一步接入 FBX/glTF 骨骼、蒙皮权重、动画极限姿态测试和团队项目数据源。
